## Nova Lands - Crafting Utility

A small utility to help you with your crafting endeavors in [Nova Lands](https://store.steampowered.com/app/1501610/Nova_Lands/). It offers following features:

- provide general information about given item
- get a list of ingredients needed to craft items
- get a list of crafting tables neccessary to craft items
- calculate minimum number of crafting tables needed for a recipe

The utiliy does not feature a graphical interface, interaction is via command prompt only.

### Installation

There are 2 options available:

#### 1. Download compiled version

Ready to run application, just download the following [.zip](./utility.zip) file. After extracting it, start the executable (.exe) file located within and you're good to go.

(The compiled version was built for Windows 11 and probably will not function with other OS, though you are welcome to try)

#### 2. Use local version of Python

#### Requirements:

- Python 3.12.10+

Clone this reposiory or download the [source code](./source.zip). In the root folder of the source code execute the following:

```
python main.py
```
